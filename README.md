**Jay Tech Payroll Pro** is a lightweight, mobile-first utility designed to simplify complex payroll calculations. Inspired by professional accounting sheets, it automates the grouping, totaling, and exporting of employee data into clean Excel reports.

## 🛠 Features
- **Zero-Config Logic:** Select a rank and let the app handle the rates.
- **Manual Overrides:** Tap any rate to adjust it for unique monthly scenarios.
- **Formula Control:** Toggle deductions (Welfare, Loan, Ration) on/off instantly.
- **Automated Reporting:** Exports grouped Excel files with sub-totals for each rank.
- **Brand Identity:** Customizable business name and gallery-loaded logo.
- **Offline First:** Built as a PWA—works on the field without internet.

## 📱 Installation (PWA)
1. Open the [Live Demo Link](YOUR_GITHUB_PAGES_URL_HERE).
2. Chrome (Android): Tap the three dots ⋮ and select **"Install App"**.
3. Safari (iOS): Tap the Share icon and select **"Add to Home Screen"**.

## 🏗 Setup & Publishing
1. Upload `index.html`, `style.css`, `script.js`, `manifest.json`, and `sw.js` to your repository.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, set the source to `Deploy from a branch` and select `main`.
4. Your app will be live at `https://yourusername.github.io/repository-name/`.
